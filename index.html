The problem

Engineers running Fitness-for-Service assessments today rely on expensive desktop software, brittle spreadsheets, or manual calculations against a 700-page standard. Every assessment takes hours, audit trails are inconsistent, and verification against API 579 Annexes is rarely documented.

What OpenFFS does

Part 4 - General Metal Loss

Cylindrical, spherical, conical, torispherical, and ellipsoidal pressure components. Effective thickness, required thickness, RSF, MAWP back-calculation, remaining life.

Part 5 - Local Metal Loss

Gouges and pits on pipework. Folias factor for long axial damage, interaction rules for multiple gouges, suitability verdicts with grind/plug weld recommendations.

Part 14 - Structural Integrity

Beams, columns, braces, angles, plates. Hardness-based strength estimates, effective section properties, stress concentration factors, Levels 1 & 2 verdicts including replace and reinforce.

Audit-ready reports

Every assessment produces a calculation log with equation IDs, substituted values, and intermediate results. Verbatim API 579 clause citations on every result.

Standards currency notifier

Tracks API 570, 571, 579, 510 and ASME B31.3, B31.1, VIII-1. Alerts you when a new edition is due for review.

CLI + Python API

Run from terminal, embed in pipelines, or build a custom GUI on top of the Python API. Your data never leaves your machine.

Get started in 5 minutes

Option 1 - Python install (recommended)

git clone https://github.com/srisurya59-art/openffs.git
cd openffs
pip install -e .
openffs --version

Option 2 - Standalone installer (Windows)

Download the bundled installer - includes Python runtime, no separate install needed.

Download OpenFFS v0.7.0 for Windows
(~80 MB - includes Python runtime)

Quick start with the example

cd examples/pipe_support_inspection
openffs assess --level 1
openffs report --format html --output report.html

The example runs an FFS-1 assessment on a 54-inch pipe support structure with 8 components. You should see 4 verdicts immediately.
