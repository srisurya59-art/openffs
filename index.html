Operational & Compliance Overview

FFS Assessment Levels Explained

Fitness-for-Service assessments per API 579-1/ASME FFS-1 follow a 3-level progressive evaluation:

  • Level 1 (Screening): Simplified calculation using nominal dimensions and conservative assumptions. Quick pass/fail check. If this calculator shows FAIL, escalate to Level 2.
  • Level 2 (Refined Analysis): Uses actual measured geometry, refined material properties at operating temperature, and stress concentration factors. Requires more detailed input data.
  • Level 3 (Detailed Analysis): Full engineering critical assessment including FEA (Finite Element Analysis), validation testing, and fracture mechanics. Required when Level 2 cannot demonstrate fitness.

Standards Foundation

All calculations follow the methodology of API 579-1/ASME FFS-1 (2021 Edition), referenced by:

  • API 570 — Piping Inspection Code
  • API 510 — Pressure Vessel Inspection Code
  • API 653 — Tank Inspection, Repair, Alteration, Reconstruction
  • ASME B31.3 / B31.1 — Process / Power Piping
