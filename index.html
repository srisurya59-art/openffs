Where OpenFFS applies

API 579-1/ASME FFS-1 is the unified Fitness-for-Service standard for process plant assets. OpenFFS implements three of its most-used Parts, covering every major asset class in your facility.

PRESSURE VESSELS

Vessels, reactors, heat exchangers

Cylindrical shells, heads (torispherical, ellipsoidal), cones, nozzles. Covers general metal loss, localized damage, and structural integrity of supports and skirts.

STORAGE TANKS

Atmospheric and low-pressure tanks

Shell plates, roof structures, nozzles, and wind girders. Supports API 653 inspection findings with FFS assessments per API 579 Part 4 and Part 14.

PIPING

Process and utility piping

Straight pipe, elbows, branch connections. Covers uniform metal loss (Part 4), local metal loss with Folias factor (Part 5), and structural support members (Part 14).

STRUCTURAL

Beams, columns, braces, plates, gussets

Pipe racks, equipment supports, bracing systems, building structures. Part 14 covers damage from impact, corrosion, overload, and fabrication defects.

One consistent methodology across every asset type — no switching between different tools for pressure equipment vs. structural steel.

What OpenFFS does

Pressure vessels, piping, storage tanks

Part 4 - General Metal Loss

Cylindrical, spherical, conical, torispherical, and ellipsoidal components. Effective thickness, required thickness, RSF, MAWP back-calculation, remaining life.

Piping, pressure equipment

Part 5 - Local Metal Loss

Gouges and pits. Folias factor for long axial damage, interaction rules for multiple gouges, suitability verdicts with grind and plug weld recommendations.

Beams, columns, braces, plates, structural supports

Part 14 - Structural Integrity

Damaged structural members. Hardness-based strength estimates, effective section properties, stress concentration factors, Levels 1 & 2 verdicts including replace and reinforce.

Audit-ready reports

Every assessment produces a calculation log with equation IDs, substituted values, and intermediate results. Verbatim API 579 clause citations on every result.

Standards currency notifier

Tracks API 570, 571, 579, 510 and ASME B31.3, B31.1, VIII-1. Alerts you when a new edition is due for review.

CLI + Python API

Run from terminal, embed in pipelines, or build a custom GUI on top of the Python API. Your data never leaves your machine.

Get started in 5 minutes

Option 1 - Python install (recommended)

git clone https://github.com/srisurya59-art/openffs.git
cd openffs
pip install -e .
openffs --version

Option 2 - Standalone installer (Windows)

Download the bundled installer - includes Python runtime, no separate install needed.

Download OpenFFS v0.7.0 for Windows
(~80 MB - includes Python runtime)

Quick start with the example

cd examples/pipe_support_inspection
openffs assess --level 1
openffs report --format html --output report.html

The example runs an FFS-1 assessment on a 54-inch pipe support structure with 8 components. You should see 4 verdicts immediately.
