# OpenFFS

Open-source Fitness-for-Service (FFS) engineering platform implementing
API 579-1/ASME FFS-1 (2021 Edition).

## Tracks

- **Part 4** - General Metal Loss (Level 1)
- **Part 5** - Local Metal Loss (Level 1)
- **Part 14** - Structural Integrity (Levels 1 & 2)

## Quick Start

pip install -e .
cd examples/pipe_support_inspection
openffs assess --level 1
openf
